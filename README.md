# Day 4 - Flow Builder Basics

## Task 1 - Automation Thinking

### 5 Automated Processes:
1. Auto email after registration
2. Auto update remaining seats
3. Notify faculty when course full
4. Generate student ID automatically
5. Send reminder before fee deadline

### Why Automation Helps:
Automation saves time, reduces errors
and ensures consistency in processes.

## Task 2 - Flow Design

### Process: Auto Email After Registration

Trigger: New Student Record Created
Step 1: Get Student Details
Step 2: Check Email exists?
Step 3: Send Welcome Email
Step 4: Create Admin Task
Final: Update Student Status = Active

## Task 3 - Manual vs Automated

### Fee Reminder Process:

Manual Way:
- Admin checks dates manually
- Types emails one by one
- Takes 3 hours daily
- Errors possible

Automated Way:
- Flow checks dates automatically
- Sends emails to all students
- Takes 0 minutes
- Zero errors

## Task 4 - Reflection

### Why Automate?
1. Saves time - works in seconds
2. No human errors
3. Works 24/7
4. Saves money
5. Better student experience
6. Staff focuses on important work
